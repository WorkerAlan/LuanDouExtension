## Installation

LuanDouExtension is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LuanDouExtension', :git => 'https://github.com/541278903/LuanDouExtension.git'
```
